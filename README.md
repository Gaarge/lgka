# Современная версия сайта ЛГКА

Файлы:

- `index.html` — структура страницы и весь текстовый контент.
- `styles.css` — адаптивный дизайн, карточки, сетки, кнопки, мобильное меню.
- `script.js` — мобильная навигация, анимация появления блоков, демо-обработка формы.
- `server.py` — локальный сервер для просмотра сайта.

## Запуск локально

```bash
python server.py
```

Затем откройте `http://127.0.0.1:8000`.

## Что заменить перед публикацией

1. Подставить реальные фотографии адвокатов вместо букв-аватаров.
2. Подключить backend-обработчик формы или сервис отправки заявок на почту.
3. При необходимости добавить отдельную страницу политики конфиденциальности.
4. Проверить актуальность телефонов и email перед публикацией.


## Как обновлять новости

Новости вынесены в отдельный файл `news-data.js`. Чтобы добавить публикацию, скопируйте объект внутри массива `newsItems`, измените `category`, `label`, `date`, `title`, `text`, `link` и `linkText`.

Доступные категории фильтра:

- `college` — новости коллегии;
- `chamber` — новости Адвокатской палаты Липецкой области;
- `fpa` — новости ФПА РФ.


## Клиенты

Раздел «Клиенты» находится в `index.html`, блок `<section id="clients">`. Сейчас логотипы сделаны как адаптивные текстовые карточки, чтобы сайт не зависел от отдельных файлов изображений. Если появятся официальные логотипы в хорошем качестве, их можно заменить на `<img>` внутри `.client-card`.
